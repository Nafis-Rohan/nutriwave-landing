# NutriWave — Responsive Healthy Meal Landing Page

NutriWave is a clean, modern, **fully responsive** landing page for a healthy-meal subscription service.  
It’s built as a front-end UI project with a focus on layout, spacing, typography, and mobile-first responsiveness.

## Link

> https://nafis-rohan.github.io/nutriwave-landing/
> https://nutriwave-nafis.netlify.app/

## Features

- ✅ **Responsive design** (desktop → tablet → mobile)
- ✅ Hero section with CTA buttons (Start / Learn more)
- ✅ “How it works” steps section with phone mockups
- ✅ Meals/recipes section + diet list (5,000+ recipes style)
- ✅ Testimonials + image gallery section
- ✅ Pricing cards (Starter / Complete)
- ✅ Benefits section (Local & organic, No waste, Pause anytime, etc.)
- ✅ Footer with useful links

## Tech Stack

- **HTML5**
- **CSS3** (Flexbox / Grid, responsive breakpoints)
- **JavaScript (Vanilla)
